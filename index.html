Import / Export.

Import problem data:

To import data, paste the generated data set in its original form. The system will automatically verify its correctness and compatibility. If any issues are found, you will be notified with details to help resolve them. Once the data is verified, simply click the button below to complete the import. Your data will be handled securely and efficiently throughout the process.

Export problem data:

After clicking the button below, the system will automatically generate the appropriate data set in the correct format. This data is ready for immediate use or future reference. To ensure it can be reused later, make sure to copy and store it in its original, unchanged form. Keeping the data intact will allow for a smooth re-import process whenever needed, without any issues or loss of information.
